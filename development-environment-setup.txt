Setting up a development environment
------------------------------------

1. Install the "Eclipse Java EE IDE for Web Developers" distribution of Eclipse, available for Mac OSX from

http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/mars/2/eclipse-jee-mars-2-macosx-cocoa-x86_64.tar.gz

Version: Mars.2 Release (4.5.2)
Build id: 20160218-0600

2. A number of projects require the Xtend editor to be installed. For this the Xtext SDK can be used. Download it from https://eclipse.org/Xtext/download.html

3. Switch to a new empty workspace

4. Import all projects from the cell repository into the empty workspace

5. To run the tests, run the JUnit .launch files in the root directories of the following projects:

org.s23m.cell.communication
org.s23m.cell.kernel
org.s23m.cell.kernel.tests
org.s23m.cell.platform

Each one can be run by right clicking it and selecting the JUnit menu item from the "Run As" menu.
